# Tweak Tool for RootHide

这是上游 Tweak_Tool 的 Dopamine2 RootHide（arm64e）适配分支。

## 构建

```sh
./scripts/build.sh
```

## 使用

安装 deb 后，在 NewTerm 中执行：

```sh
sudo tweaktool
```

或在 Filza 中运行：

```text
/var/mobile/Documents/tweak_tool/一键备份和恢复工具.sh
```

上游项目及原作者：[Ketty0725/Tweak_Tool](https://github.com/Ketty0725/Tweak_Tool)。
